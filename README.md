# 📦 Logistics Management Database

## Overview
This project contains a relational database schema designed for managing logistics operations.  
It models companies, tasks, trucks, trailers, drivers, and garages with realistic attributes and relationships.

## Features
- **Normalized relational design** with foreign keys for data integrity.  
- **Driver ranking system**: Automatically assigns ranks (Beginner → Veteran) and pay rates per km based on distance driven.  
- **Garage status view** (`GarageStatus`): Summarizes the number of trucks and trailers per garage.  
- **Sample data**: Includes companies, tasks, trucks, drivers, and trailers for demonstration.  

## Use Cases
- Logistics company management  
- Fleet tracking and driver performance analysis  
- Academic projects and SQL practice  

## Technologies
- MySQL / MariaDB  
- SQL Views & Constraints  
- Relational Database Design

# 📦 Lojistik Yönetim Veritabanı

## Genel Bakış
Bu proje, lojistik operasyonlarını yönetmek için tasarlanmış ilişkisel bir veritabanı şeması içerir.  
Şirketler, görevler, TIR’lar, dorseler, şoförler ve garajlar gerçekçi özellikler ve ilişkilerle modellenmiştir.

## Özellikler
- **Normalleştirilmiş ilişkisel tasarım** ve yabancı anahtarlarla veri bütünlüğü sağlanır.  
- **Şoför derecelendirme sistemi**: Kat edilen kilometreye göre otomatik rütbe (Acemi → Duayen) ve km başına ücret atanır.  
- **Garaj durumu görünümü** (`GarageStatus`): Her garajdaki TIR ve dorse sayısını özetler.  
- **Örnek veriler**: Şirketler, görevler, TIR’lar, şoförler ve dorseler için eklenmiş test verileri bulunur.  

## Kullanım Alanları
- Lojistik şirket yönetimi  
- Filo takibi ve şoför performans analizi  
- Akademik projeler ve SQL pratikleri  

## Teknolojiler
- MySQL / MariaDB  
- SQL Görünümleri & Kısıtlamalar  
- İlişkisel Veritabanı Tasarımı
